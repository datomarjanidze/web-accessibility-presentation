Input Labels

<div>First Name</div>
<input type="text" />
First Name
  1. <input> element should not have other element than <label> for the purpose of labeling, because assistive technologies like screen readers won't be able to tell the input's label
<label for="first-name">First Name</label>
<input id="first-name" type="text" />
<br />
<label class="last-name" >Last Name <input type="text" /> </label>

  1. Assistive technologies like screen readers can read the label out loud when the user is focused on the input
  2. The <label> element must have `for` attribute set to the id of the input element to link them together, or you can place the input element into the label tag
